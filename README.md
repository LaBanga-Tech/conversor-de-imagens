## BIBLIOTECAS QUE DEVEM SER INSTALADAS
- CustomTkinter
- Pillow

## COMO INSTALAR?

```bash
    pip install customtkinter pillow
```

## COMO RODAR O PROGRAMA?

```bash
    py main.py
```

*NOTA: Este app não converte outras extensões para jpg/jpeg*
